## Installation

---
1. Download the zip archive LRPrintEdition-1.0.0.0.zip from 
   [GitHub](https://github.com/sto3014/LRPrintEdition/archive/refs/tags/1.0.0.0.zip).
2. Extract the archive in the download folder 
3. Open a terminal window, change to Downloads/LRPrintEdition-1.0.0.0 and execute install.sh:
    ```
   -> ~ cd Downloads/LRPrintEdition-1.0.0.0
   -> ./install.sh 
    ```
    Install.sh extracts the plug-in into:
    ```
    ~/Library/Application Support/Adobe/Lightroom/Modules/LRPrintEdition.lrplugin
    ```
4. Restart Lightroom
